# PyDSAI

AI 赋能的数据结构与算法实现。

## 项目结构

```
PyDSAI/
├── src/
│   └── pydsai/      # 核心库代码
├── tests/           # 测试代码
└── README.md
```

## 开发环境

- Python 3.11+
- 代码风格：Black (line length 88)
- 类型检查：mypy strict mode
- 测试：pytest + pytest-cov

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```python
import pydsai

print(pydsai.__version__)
```
